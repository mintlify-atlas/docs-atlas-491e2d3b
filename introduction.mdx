## What is LiquidLauncher?

LiquidLauncher is a powerful, cross-platform desktop application built with Tauri that makes launching LiquidBounce for Minecraft effortless. It handles everything from account authentication to mod management, version control, and Java runtime configuration.

## Key Features

<AccordionGroup>
  <Accordion title="Microsoft & Offline Authentication" icon="shield-check">
    LiquidLauncher supports both Microsoft account authentication and offline play. Sign in with your Microsoft account for full Minecraft features, or create an offline account for local play. Manage multiple accounts and switch between them effortlessly.
  </Accordion>
  
  <Accordion title="Automatic Updates" icon="arrow-up-right-dots">
    The launcher automatically checks for and downloads updates for both itself and LiquidBounce builds. You'll always have the latest features and bug fixes without manual intervention.
  </Accordion>
  
  <Accordion title="Custom Mod Management" icon="box-open">
    Install custom mods from local JAR files and manage them directly from the launcher. The mod system supports per-version and per-branch configurations, making it easy to customize your experience.
  </Accordion>
  
  <Accordion title="Java Runtime Management" icon="coffee">
    LiquidLauncher automatically downloads and configures the correct Java runtime for your game version. No need to manually install Java or worry about compatibility issues.
  </Accordion>
  
  <Accordion title="Version & Build Selection" icon="code-branch">
    Choose from different LiquidBounce branches (stable, beta, nightly) and specific builds. Each version is automatically configured with the correct Minecraft version and dependencies.
  </Accordion>
  
  <Accordion title="Cross-Platform Support" icon="desktop">
    Built with Tauri and Rust, LiquidLauncher runs natively on Windows, macOS, and Linux with excellent performance and a modern, responsive UI.
  </Accordion>
</AccordionGroup>

## Getting Started

<Steps>
  <Step title="Download LiquidLauncher">
    Visit the [installation page](/installation) to download the latest version for your operating system.
  </Step>
  
  <Step title="Install and Launch">
    Run the installer and launch LiquidLauncher. The application will automatically set up the necessary directories and configuration.
  </Step>
  
  <Step title="Sign In">
    Sign in with your Microsoft account or create an offline account to get started.
  </Step>
  
  <Step title="Launch the Game">
    Select your preferred LiquidBounce version and click **Launch** to start playing!
  </Step>
</Steps>
